## 💡 使用技巧

### 最佳使用场景
- 📚 **教育视频**：快速了解课程内容结构
- 🎤 **演讲/讲座**：提取演讲的核心论点
- 📰 **新闻/访谈**：快速获取关键信息
- 🛠️ **技术教程**：了解教程的主要步骤

### 注意事项
1. **字幕质量**：
   - 自动生成的字幕可能有错误，会影响总结质量


2. **视频长度**：
   - 适用于 5-60 分钟的视频
   - 太短的视频总结意义不大
   - 太长的视频可能只分析前 8000 字符的内容

3. **语言设置**：
   - 总结会使用您在设置中选择的目标语言
   - 例如选择"简体中文"，无论原视频什么语言，总结都会用中文呈现

4. **API 费用**：
   - 总结功能会消耗较多 token（约 500-2000）
   - 建议选择性使用，不要对每个视频都生成总结

**解决方案**：

- 确认 API Key 有足够的余额

**解决方案**：
- 检查视频是否有高质量字幕
- 尝试使用更高级的 AI 模型
