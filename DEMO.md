# 使用演示流程

## 场景：在 Chrome 中安装并使用扩展

### 步骤 1：安装扩展（2 分钟）

```
1. 打开 Chrome 浏览器
2. 地址栏输入：chrome://extensions/
3. 右上角启用"开发者模式"开关
4. 点击左上角"加载已解压的扩展程序"
5. 浏览到并选择 youtube-translator-extension 文件夹
6. 点击"选择文件夹"

✅ 扩展已安装！工具栏会出现扩展图标
```

### 步骤 2：配置 API（3 分钟）

#### 方式 A：使用 OpenAI

```
1. 访问 https://platform.openai.com/api-keys
2. 登录账号（如无账号需注册）
3. 点击 "Create new secret key"
4. 复制生成的 API Key（sk-...）
5. 在 Chrome 点击扩展图标
6. 点击"⚙️ 配置 API 设置"
7. 选择 "OpenAI" 单选按钮
8. 粘贴 API Key
9. 选择模型：GPT-4o-mini（推荐新手）
10. 选择目标语言：简体中文
11. 点击"保存设置"

✅ OpenAI 配置完成！
```

#### 方式 B：使用 Claude

```
1. 访问 https://console.anthropic.com/settings/keys
2. 登录账号（如无账号需注册）
3. 点击 "Create Key"
4. 复制生成的 API Key（sk-ant-...）
5. 在 Chrome 点击扩展图标
6. 点击"⚙️ 配置 API 设置"
7. 选择 "Claude (Anthropic)" 单选按钮
8. 粘贴 API Key
9. 选择模型：Claude 3.5 Haiku（推荐新手）
10. 选择目标语言：简体中文
11. 点击"保存设置"

✅ Claude 配置完成！
```

### 步骤 3：翻译视频（1 分钟）

```
1. 打开 YouTube 任意视频
   例如：https://www.youtube.com/watch?v=dQw4w9WgXcQ

2. 确保视频有字幕
   - 点击视频右下角的 CC 字幕按钮
   - 如已开启会显示字幕

3. 点击 Chrome 工具栏的扩展图标

4. 点击"开始翻译"按钮

5. 播放视频
   - 字幕出现时，翻译会自动显示在视频下方
   - 黑色半透明背景，白色文字
   - 翻译会随字幕更新

6. 停止翻译
   - 再次点击扩展图标
   - 点击"停止翻译"

✅ 翻译完成！
```

## 高级技巧

### 1. 切换模型以平衡质量和成本

**追求质量：**
- OpenAI: GPT-4o
- Claude: Claude 3.5 Sonnet

**追求速度和经济：**
- OpenAI: GPT-4o-mini 或 GPT-3.5-turbo
- Claude: Claude 3.5 Haiku

### 2. 多语言支持

```
英语视频 → 中文翻译：最常见
日语视频 → 中文翻译：动漫、日剧
中文视频 → 英语翻译：学习英语
韩语视频 → 中文翻译：韩剧、韩综
```

### 3. 性能优化

- 字幕更新频繁时，选择快速模型
- 专业内容（技术、学术），选择高质量模型
- 测试时先用 mini/haiku 确保配置正确

## 故障排除

### ❌ 问题：点击"开始翻译"没反应

**解决方案：**
1. 刷新 YouTube 页面（F5）
2. 确保字幕已开启（点击 CC 按钮）
3. 打开控制台（F12）查看错误信息

### ❌ 问题：提示"请配置 API Key"

**解决方案：**
1. 点击"⚙️ 配置 API 设置"
2. 确认已选择提供商并输入有效的 API Key
3. 点击"保存设置"

### ❌ 问题：翻译延迟很高

**解决方案：**
1. 切换到更快的模型（mini/haiku）
2. 检查网络连接
3. 确认 API 服务状态正常

### ❌ 问题：API 调用失败

**解决方案：**
1. 检查 API Key 是否正确
2. 确认账户余额充足
3. 查看 API 服务状态页面

## 费用估算

### 典型场景：观看 1 小时英文视频

**假设：**
- 字幕更新频率：每 5 秒一次
- 每次翻译约 50 tokens
- 总计：~720 次请求 × 50 tokens = 36,000 tokens

**成本：**
- GPT-4o-mini: ~$0.005（半分钱）
- GPT-4o: ~$0.18（约 1.3 元）
- Claude 3.5 Haiku: ~$0.009（约 0.06 元）
- Claude 3.5 Sonnet: ~$0.11（约 0.75 元）

💡 **结论：使用经济型模型，每小时成本不到 1 分钱！**
