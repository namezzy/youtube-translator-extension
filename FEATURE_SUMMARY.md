# 视频内容总结功能

## 功能概述

FEATURE_SUMMARY. YouTube 视频字幕，自动生成结构化的内容总结。

## 主要特性

### 1. 智能内容分析
- 自--------心观点
- 将内容划分为 3-5 个主要部分
- 每个部分包含标题和核心要点

### 2. 多语言支持
- 支持所有已配置的目标语言
- 与翻译功能使用相同的语言设置

### 3. 全平台兼容
- 支持所有 AI 提供商（OpenAI、Claude、Grok、Groq、Gemini、自定义 AI）
- 自动适配不同 API 格式

## 使用方法

1. 在 YouTube 视频页面点击插件图标
2. 点击 **"📝 总结视频内容"** 按钮
cd /root/p_website && git push 
4. 总结会显示在视频播放器下方
